# MigrosMoneyKartUygulama
JavaScript Bu proje, JavaScript içinde sınıflar kullanılarak geliştirilmiş bir Migros Money Kart uygulamasıdır. Uygulama, kullanıcıların Migros kartlarına sahip olup olmadıklarını kontrol eder ve kart sahiplerine indirimli fiyatlarla ürünler sunar. Kullanıcılar, ürünlerin normal fiyatlarını ve indirimli fiyatlarını görebilirler.
